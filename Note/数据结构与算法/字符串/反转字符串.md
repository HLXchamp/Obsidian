用**双指针法**就好！head指向头，tail指向尾，注意声明方法`char t = '\0';`

```c
void reverseString(char* s, int sSize) {
    int head = 0;
    int tail = sSize-1;
    while(head < tail){
        char t = '\0';
        t = s[tail];
        s[tail] = s[head];
        s[head] = t;
        head++;
        tail--;
    }
}
```
