**提示：**
- 树中节点的数目范围是 `[2, 104]`
- `0 <= Node.val <= 10^5`

### 迭代法

和上一道题[[验证二叉搜索树#递归法]]思路差不多~因为前序遍历后就是一个递增的数组，只需要遍历的过程中进行相减和判断就行。


```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    int min = 100000;
    TreeNode prev = null;
    public int getMinimumDifference(TreeNode root) {
        if(root == null){
            return min;
        }
        getMinimumDifference(root.left); // 左
        if(prev != null && root.val - prev.val <= min){ // 中
            min = root.val - prev.val;
        }
        prev = root;
        getMinimumDifference(root.right); // 右
        return min;
    }
}
```

注：
- min初始应设的**越大越好**，也可以设为`INT_MAX`
