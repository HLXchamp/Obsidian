
中序遍历：

```java
class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if(root == null){
            return res;
        }
        Stack<TreeNode> stack = new Stack<>();
        TreeNode cur = root;
        while(cur != null || !stack.isEmpty()){
            if(cur != null){
                stack.push(cur);
                cur = cur.left; // 向左走到头
            }
            else{
                cur = stack.pop(); // 到头就可以出栈入数组了
                res.add(cur.val);
                cur = cur.right;
            }
        }
        return res;
    }
}
```
